# Instalaciones

## 1. Introducción
Este manual está diseñado para guiar a los usuarios en la creación y planificación de instalaciones y montajes desde ENBLAU. De esta manera, se pueden asignar a los montadores internos y externos las instalaciones planificadas desde ENBLAU.

---

## 2. Instalaciones

En Instalaciones se puede acceder a **Instalación**, **Instaladores externos** y el **Planificador de instalación**.

  ![Instalaciones](Imagenes/PR_Instalaciones/instalaciones.jpg) 

### 2.1. Instalación

#### 2.1.1. Configuraciones

Desde el apartado de Instalaciones - **Instalación** para definir las **Configuraciones**.

  ![Instalacion configuracion](Imagenes/PR_Instalaciones/instalacion_configuracion.jpg)

**- 1. Numeraciones** 
- En Numeraciones definir las numeraciones para cada tipo de documento.

    ![Instalacion numeraciones](Imagenes/PR_Instalaciones/instalacion_numeraciones.jpg)

**- 2. Clasificaciones** 
- Definir las clasificaciones según necesario para asignar al documento de instalación.

    ![Instalacion clasificaciones](Imagenes/PR_Instalaciones/instalacion_clasificacion.jpg)

**- 3. Planificación** 
- Definir en Planificación las **Clasificaciones** y **Estados**.

    ![Instalacion clasificaciones](Imagenes/PR_Instalaciones/instalacion_plan_clasificaciones.jpg)

    ![Instalacion clasificaciones](Imagenes/PR_Instalaciones/instalacion_plan_clasificaciones2.jpg)

    ![Instalacion clasificaciones](Imagenes/PR_Instalaciones/instalacion_plan_estados.jpg)

    ![Instalacion clasificaciones](Imagenes/PR_Instalaciones/instalacion_plan_estados2.jpg)

    - Las **Clasificaciones** y **Estados** al crear una nueva tarea en el apartado del **Planificador de instalación**. 


**- 4. Directorios por defecto**

- Al seleccionar **Directorios por Defecto**, se abrirá una ventana para definir los directorios necesarios en la carpeta de instalación.

    ![Instalacion directorios](Imagenes/PR_Instalaciones/instalacion_dir.jpg)

    ![Instalacion directorios](Imagenes/PR_Instalaciones/instalacion_dir2.jpg)

> ℹ️ **Nota:** Los directorios predeterminados son carpetas que pueden configurarse para que se generen automáticamente con cada instalación.


**- 5. Definición de estados**

- Al seleccionar **Definición de estados**, se abrirá una ventana para definir las la definición de estados para cada tipo de documento.

    ![Instalacion estados](Imagenes/PR_Instalaciones/instalacion_estados.jpg)

#### 2.1.2. Crear instalación 

**- 1. Nuevo** 
- Para crear una instalación ir al apartado de Instalación - **Nuevo**. Abrirá una ventana de Nuevo documento de instalación donde hay que indicar algunos campos como Tipo de documento, Numeración, Numeración de documento, clasificación, instalador externo y proyecto.

    ![Instalacion nuevo](Imagenes/PR_Instalaciones/instalacion_nuevo.jpg)

**- 2. Documento de instalación** 

-   **Detalle**
    En la pestaña *detalle* del documento de instalación aparece todos los campos rellenados previamente, pero que es editable. Luego en la parte inferior se puede añadir posiciones y definir el nombre del artículo, descripción, unidades, precio, etc. 

    ![Instalacion nuevo](Imagenes/PR_Instalaciones/instalacion_nuevo2.jpg)

-   **Estados y observaciones**
    En la pestaña *estados y observaciones* indica algunos campos como forma de pago, fechas de solicitud y cancelación, IVA, etc. 

    ![Instalacion nuevo](Imagenes/PR_Instalaciones/instalacion_nuevo5.jpg)

-   **Planificación**
    En la pestaña *planificación* del documento de instalación es donde se indica las fechas de inicio y fin para la planificación y asignar instaladores interno.

    ![Instalacion nuevo](Imagenes/PR_Instalaciones/instalacion_nuevo3.jpg)

-   **Observaciones para instalación**
    En la pestaña *observación para instalación* del documento de instalación es donde se indica las fechas de inicio y fin para la planificación y asignar instaladores interno.

    ![Instalacion nuevo](Imagenes/PR_Instalaciones/instalacion_nuevo3.jpg)

#### 2.1.3. Crear instalación desde un documento de venta
- Desde un documento de venta (pedido, medición o producción) se puede crear una previsión de instalación o crear una instalación.

    ![Instalacion pedido](Imagenes/PR_Instalaciones/instalacion_pedido.jpg) 

**- 1. Previsión de instalación** 
    En ese apartado se puede indicar una previsión de instalación indicando el instalador externo, fecha estimada e importes pactados.

    ![Instalacion pedido](Imagenes/PR_Instalaciones/instalacion_pedido2.jpg) 

**- 2. Crear Instalación** 
    Al crear una instalación desde un documento de ventas se debe indicar las unidades a instalar de cada posición del documento en la instalación. Luego se puede añadir instaladores externos o internos a esa instalación.

    ![Instalacion pedido](Imagenes/PR_Instalaciones/instalacion_pedido3.jpg)

#### 2.1.4. Imprimir informes de instalación 
- Imprimir los informes disponibles de instalación, Parte de instalación, Partes de tiempos y Pedido de instalación. Así como cualquier informe de un documento, antes de imprimir el informe tienes las condiciones y una información breve sobre el informe seleccionado.

    ![Instalacion imprimir](Imagenes/PR_Instalaciones/instalacion_imprimir.jpg)

### 3.1. Instaladores externos
- Para dar de alta a los instaladores externos, ir al apartado de Instalación - **Instaladores externos - Nuevo.** Desde ahí rellenar los campos oportunos como Nombre, Dirección, NIF, Datos de contacto, Precios, etc.  

    ![Instalacion instalador externo](Imagenes/PR_Instalaciones/instalacion_inst_ext.jpg)

---

### 4.1. Planificador de instalaciones
- En el apartado de Instalaciones - **Planificador de Instalaciones** puedes ver y controlar las tareas desde un calendario para cada instalador (interno o externo).  

    ![Instalacion planificador](Imagenes/PR_Instalaciones/instalacion_planificador.jpg)

**- 1. Crear tarea** 
- Para crear una nueva tarea desde el planificador seleccionar **Nueva tarea**. Abrirá una ventana *Nuevo evento de instalación* donde se asigna el instalador, indica el número de instalación y el proyecto.  

    ![Instalacion planificador](Imagenes/PR_Instalaciones/instalacion_planificador2.jpg)

- Luego te abrirá una ventana de *Tarea* donde puedes indicar estados, clasificaciones, fecha y hora de inicio y fin, añadir descripción corta y notas la tarea. 

    ![Instalacion planificador](Imagenes/PR_Instalaciones/instalacion_planificador3.jpg)

**- 2. Calendario**
- Desde el calendario verás reflejado la tarea asignada al instalador donde al pasar el ratón por encima te mostrara una información mínima como nombre del proyecto, número de instalación y descripción corta. Los colores indican estado y clasificación (previamente definido en el apartado de configuración Planificador). 

    ![Instalacion planificador](Imagenes/PR_Instalaciones/instalacion_planificador4.jpg)
